> # DB에 데이터가 남아있을 시에 
>> ```
>> ALTER TABLE [테이블 이름] AUTO_INCREMENT=1;
>> SET @COUNT = 0;
>> UPDATE [테이블 이름] SET [AI가 적용된 Column] = @COUNT:=@COUNT+1;
>> ```
